# Tiktok api

Get video for TikTok without watermark, api bypass "snaptik.app" 

## Installation

Install requirement

```bash
  pip install -r requirement.txt
```

## Run Locally

Clone the project

```bash
  git clone github.com/Ghost6446/Tik_tok_api_1
```

Go to the project directory

```bash
  cd Tik_tok_api_1
```

Start scrapper

```bash
  python run.py
```
